# Fabricación de letreros 3D

Este proyecto documenta mi proceso para diseñar y fabricar letreros 3D utilizando impresión 3D multicolor con AMS.

---

# Herramientas

- Impresora 3D Bambu Lab
- AMS (Automatic Material System)
- Bambu Studio
- Filamentos PLA

---
